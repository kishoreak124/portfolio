Project

Abstract

Fishermen’s often face serious risks when they unintentionally cross international maritime boundaries due to the lack of accurate real-time navigation systems. Incidents along the India–Sri Lanka border have repeatedly arrests, penalties, and even loss of lives, making maritime boundary awareness a critical need. To resolve this main issue, this project proposes a Smart Marine Time Boundary Monitoring System for Fisherman’s safety, an intelligent, low coast, and scalable solution that integrates GPS (Global Positioning System),GSM (Global System For Mobile Communications), and microcontroller-based geo-fencing technology. The system continuously monitors the location of the fishing boats in time using GPS coordinates and defined a virtual marine time boundary. Whenever a fishing boat approaches or crosses the defined geo-fence ,the system immediately triggers multi-level alerts through buzzer, siren, and SMS notifications to both fisherman’s and nearby coast guard. A visual interface is also provided using an interactive map to track and log all movements. Unlike conventional GPS systems that only display coordinates, the proposed system offers active decision-based alerting, real-time monitoring, ensuring safety even in remote sea zones with limited connectivity. The system’s sea only accuracy minimizes land- based errors and ensures reliable boundary detection. This innovation marks a major move towards avoiding cross-border incidents, improving fisherman’s awareness, and strengthening maritime security. It closes the divide the gap between basic navigation system and advanced maritime surveillance, providing socially impactful and technologically viable for large-scale implementations in coastal communities.
